Helpful Resources

Angular Angular is a front-end framework maintained by Google. This website uses version 4.
Angularfire2 Angularfire2 is a library that makes it easy for Angular apps to interact with Google's Cloud Firestore or Firebase.
Firebase For deployment, I used Firebase Hosting. Firebase is a Backend-as-a-Service — BaaS — that started as a YC11 startup and grew up into a next-generation app-development platform on Google Cloud Platform.
Firestore For data management, I'm using Google's new Cloud Firestore. According to Google, Cloud Firestore is a fully-managed NoSQL document database for mobile and web app development. It's designed to easily store and sync app data at global scale, and it's now available in beta.
PrimeNG PrimeNG is an open-source collection of UI components designed for Angular.
Font Awesome Font Awesome is an open-source collection of scalable vector icons.
